# Basalt — Ultra-High Performance KV Store for AI Memory

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** Build the fastest in-memory KV store purpose-built for AI agent memory, with both HTTP and RESP protocols. Beat Redis/Memcached on throughput for AI memory workloads.

**Architecture:** Sharded concurrent hashtable in Rust. Two protocol frontends (HTTP via axum, RESP via custom TCP parser) backed by the same engine. Namespaces are first-class. Memory-type-aware operations.

**Tech Stack:** Rust 2024 edition, Tokio, axum (HTTP), custom RESP2 parser (TCP), papaya (concurrent hashmap), fxhash, serde/json, clap, tracing, criterion

---

## Why Basalt Can Win

Redis is single-threaded on the data path. Memcached is multi-threaded but generic. Basalt exploits AI memory workload properties:

1. **Read-heavy (50-100:1)** — papaya's lock-free reads dominate here
2. **Namespace-partitioned** — first-class namespaces, not key prefixes
3. **TTL-aware by type** — episodic auto-expires, semantic/procedural don't
4. **Small-to-medium values** — skip blob overhead
5. **Bulk retrieval** — grab all memories for an agent in one call

## Dual Protocol Design

```
                    ┌─────────────┐
    HTTP ──────────►│   axum      │
  (port 7380)      │   router    │
                    ├─────────────┤
                    │  Command    │────►  Sharded KV Engine
    RESP ──────────►│  Dispatch   │       (papaya HashMaps)
  (port 6380)      │             │
                    └─────────────┘
```

- **HTTP (port 7380)** — Primary API for AI agents. JSON bodies. RESTful namespace paths.
- **RESP (port 6380)** — Redis-compatible protocol. For tooling, benchmarks, legacy clients.
- Both talk to the same `KvEngine` — no data duplication.

## HTTP API

```
POST   /store/{namespace}                 → store a memory
  Body: { "key": "obs:1", "value": "saw a red car", "type": "episodic", "ttl_ms": 3600000 }
  Response: { "ok": true }

GET    /store/{namespace}                 → list all memories in namespace
  Query: ?type=episodic&prefix=obs
  Response: { "memories": [{ "key": "obs:1", "value": "saw a red car", "type": "episodic", "ttl_ms": 3599420 }] }

GET    /store/{namespace}/{key}           → get specific memory
  Response: { "key": "obs:1", "value": "saw a red car", "type": "episodic", "ttl_ms": 3599420 }

DELETE /store/{namespace}/{key}           → delete a memory
  Response: { "ok": true, "deleted": 1 }

DELETE /store/{namespace}                 → delete entire namespace
  Response: { "ok": true, "deleted": 47 }

GET    /health                            → health check
  Response: { "status": "ok" }

GET    /info                              → server stats
  Response: { "version": "0.1.0", "shards": 64, "namespaces": 12, "keys": 48293 }
```

Internal key format: `{namespace}:{key}` — but this is invisible to HTTP users. The namespace is peeled off the URL path.

## RESP Commands

Standard Redis-compatible:
- `SET key value [EX sec | PX ms]`
- `GET key`
- `DEL key [key ...]`
- `MGET key [key ...]`
- `MSET key value [key value ...]`
- `KEYS prefix*`
- `PING`
- `INFO`

Basalt-specific:
- `MSETT key value type [PX ms]` — set with memory type
- `MGETT key` — get value + type + TTL
- `MSCAN prefix` — scan all key-value pairs for a prefix
- `MTYPE key` — get the memory type of a key

## Memory Types

- **Episodic** — Conversations, observations, events. Auto-expires (default: 1 hour TTL).
- **Semantic** — Facts, rules, learned knowledge. No expiry.
- **Procedural** — Skills, how-to knowledge. No expiry.

```rust
use papaya::HashMap;
use std::time::{Duration, Instant};
use crate::store::memory_type::MemoryType;

pub struct Entry {
    pub value: Vec<u8>,
    pub expires_at: Option<Instant>,
    pub memory_type: Option<MemoryType>,
}

impl Entry {
    pub fn is_expired(&self) -> bool {
        self.expires_at.map_or(false, |t| Instant::now() >= t)
    }

    pub fn ttl_remaining_ms(&self) -> Option<u64> {
        self.expires_at.map(|t| {
            t.duration_since(Instant::now())
             .map(|d| d.as_millis() as u64)
             .unwrap_or(0)
        })
    }
}

pub struct Shard {
    data: HashMap<String, Entry>,
}

impl Shard {
    pub fn new() -> Self {
        Self { data: HashMap::new() }
    }

    pub fn set(&self, key: String, value: Vec<u8>, ttl_ms: Option<u64>, memory_type: Option<MemoryType>) {
        let effective_ttl = ttl_ms.or_else(|| memory_type.and_then(|mt| mt.default_ttl_ms()));
        let expires_at = effective_ttl.map(|ms| Instant::now() + Duration::from_millis(ms));
        let entry = Entry { value, expires_at, memory_type };
        self.data.pin().insert(key, entry);
    }

    pub fn get(&self, key: &str) -> Option<Vec<u8>> {
        self.data.pin().get(key).and_then(|entry| {
            if entry.is_expired() { None } else { Some(entry.value.clone()) }
        })
    }

    pub fn get_entry(&self, key: &str) -> Option<Entry> {
        self.data.pin().get(key).and_then(|entry| {
            if entry.is_expired() { None } else { Some(entry.clone()) }
        })
    }

    pub fn delete(&self, key: &str) -> bool {
        self.data.pin().remove(key).is_some()
    }

    pub fn scan_prefix(&self, prefix: &str) -> Vec<(String, Entry)> {
        self.data.pin()
            .iter()
            .filter(|(k, v)| k.starts_with(prefix) && !v.is_expired())
            .map(|(k, v)| (k.clone(), v.clone()))
            .collect()
    }

    pub fn len(&self) -> usize {
        self.data.pin().len()
    }
}

impl Clone for Entry {
    fn clone(&self) -> Self {
        Self {
            value: self.value.clone(),
            expires_at: self.expires_at,
            memory_type: self.memory_type,
        }
    }
}
```

**Step 3: engine.rs**

```rust
use crate::store::shard::{Shard, Entry};
use crate::store::memory_type::MemoryType;

pub const DEFAULT_SHARD_COUNT: usize = 64;

pub struct KvEngine {
    shards: Vec<Shard>,
    shard_mask: usize,
    shard_count: usize,
}

#[derive(Debug)]
pub struct EntryMeta {
    pub memory_type: Option<MemoryType>,
    pub ttl_remaining_ms: Option<u64>,
}

impl KvEngine {
    pub fn new(shard_count: usize) -> Self {
        let count = shard_count.next_power_of_two();
        let shards = (0..count).map(|_| Shard::new()).collect();
        Self {
            shards,
            shard_mask: count - 1,
            shard_count: count,
        }
    }

    fn shard_for(&self, key: &str) -> &Shard {
        let hash = fxhash::hash(key) as usize;
        &self.shards[hash & self.shard_mask]
    }

    /// Set a value. Key should be fully qualified (namespace:key).
    pub fn set(&self, key: &str, value: &[u8], ttl_ms: Option<u64>, memory_type: Option<MemoryType>) {
        self.shard_for(key).set(key.to_string(), value.to_vec(), ttl_ms, memory_type);
    }

    pub fn get(&self, key: &str) -> Option<Vec<u8>> {
        self.shard_for(key).get(key)
    }

    pub fn get_with_meta(&self, key: &str) -> Option<(Vec<u8>, EntryMeta)> {
        self.shard_for(key).get_entry(key).map(|entry| {
            let meta = EntryMeta {
                memory_type: entry.memory_type,
                ttl_remaining_ms: entry.ttl_remaining_ms(),
            };
            (entry.value, meta)
        })
    }

    pub fn delete(&self, key: &str) -> bool {
        self.shard_for(key).delete(key)
    }

    /// Scan all keys matching a prefix. Returns (key, value, meta) triples.
    /// Prefix should include the namespace, e.g. "agent-42:"
    pub fn scan_prefix(&self, prefix: &str) -> Vec<(String, Vec<u8>, EntryMeta)> {
        self.shards.iter()
            .flat_map(|shard| {
                shard.scan_prefix(prefix).into_iter().map(|(k, entry)| {
                    let meta = EntryMeta {
                        memory_type: entry.memory_type,
                        ttl_remaining_ms: entry.ttl_remaining_ms(),
                    };
                    (k, entry.value, meta)
                })
            })
            .collect()
    }

    /// Delete all keys matching a prefix. Returns count deleted.
    pub fn delete_prefix(&self, prefix: &str) -> usize {
        // Collect keys first to avoid holding shard locks during delete
        let keys: Vec<String> = self.shards.iter()
            .flat_map(|shard| shard.scan_prefix(prefix))
            .map(|(k, _)| k)
            .collect();
        let count = keys.len();
        for key in &keys {
            self.shard_for(key).delete(key);
        }
        count
    }

    /// Count keys matching a prefix.
    pub fn count_prefix(&self, prefix: &str) -> usize {
        self.shards.iter()
            .map(|shard| shard.scan_prefix(prefix).len())
            .sum()
    }

    pub fn shard_count(&self) -> usize {
        self.shard_count
    }
}
```

**Step 2: server.rs — axum router + handlers**

```rust
use axum::{
    Router,
    routing::{get, post, delete},
    extract::{Path, Query, State},
    http::StatusCode,
    Json,
};
use std::sync::Arc;
use crate::store::engine::KvEngine;
use crate::http::models::*;

pub struct AppState {
    pub engine: KvEngine,
}

pub fn app(engine: KvEngine) -> Router {
    let state = Arc::new(AppState { engine });
    Router::new()
        .route("/health", get(health))
        .route("/info", get(info))
        .route("/store/{namespace}", post(store_memory))
        .route("/store/{namespace}", get(list_memories))
        .route("/store/{namespace}", get(list_memories))
        .route("/store/{namespace}", delete(delete_namespace))
        .route("/store/{namespace}/{key}", get(get_memory))
        .route("/store/{namespace}/{key}", delete(delete_memory))
        .with_state(state)
}

async fn health() -> Json<HealthResponse> {
    Json(HealthResponse { status: "ok".into() })
}

async fn info(State(state): State<Arc<AppState>>) -> Json<InfoResponse> {
    Json(InfoResponse {
        version: "0.1.0".into(),
        shards: state.engine.shard_count(),
    })
}

async fn store_memory(
    State(state): State<Arc<AppState>>,
    Path(namespace): Path<String>,
    Json(body): Json<StoreRequest>,
) -> Result<Json<SimpleResponse>, StatusCode> {
    let full_key = format!("{}:{}", namespace, body.key);
    state.engine.set(
        &full_key,
        body.value.as_bytes(),
        body.ttl_ms,
        body.r#type,
    );
    Ok(Json(SimpleResponse { ok: true, deleted: None }))
}

async fn list_memories(
    State(state): State<Arc<AppState>>,
    Path(namespace): Path<String>,
    Query(query): Query<ListQuery>,
) -> Json<ListResponse> {
    let prefix = format!("{}:", namespace);
    let entries = state.engine.scan_prefix(&prefix);

    let memories: Vec<StoreResponse> = entries
        .into_iter()
        .filter(|(key, _, meta)| {
            if let Some(ref mt) = query.r#type {
                meta.memory_type == Some(*mt)
            } else {
                true
            }
        })
        .filter(|(key, _, _)| {
            if let Some(ref qp) = query.prefix {
                // Strip namespace prefix, check if the remaining key starts with query prefix
                let stripped = key.strip_prefix(&prefix).unwrap_or(key);
                stripped.starts_with(qp.as_str())
            } else {
                true
            }
        })
        .map(|(key, value, meta)| {
            let display_key = key.strip_prefix(&prefix).unwrap_or(&key).to_string();
            StoreResponse {
                key: display_key,
                value: String::from_utf8_lossy(&value).to_string(),
                r#type: meta.memory_type.map(|mt| mt.to_string()),
                ttl_ms: meta.ttl_remaining_ms,
            }
        })
        .collect();

    Json(ListResponse { memories })
}

async fn get_memory(
    State(state): State<Arc<AppState>>,
    Path((namespace, key)): Path<(String, String)>,
) -> Result<Json<StoreResponse>, StatusCode> {
    let full_key = format!("{}:{}", namespace, key);
    match state.engine.get_with_meta(&full_key) {
        Some((value, meta)) => {
            Ok(Json(StoreResponse {
                key,
                value: String::from_utf8_lossy(&value).to_string(),
                r#type: meta.memory_type.map(|mt| mt.to_string()),
                ttl_ms: meta.ttl_remaining_ms,
            }))
        }
        None => Err(StatusCode::NOT_FOUND),
    }
}

async fn delete_memory(
    State(state): State<Arc<AppState>>,
    Path((namespace, key)): Path<(String, String)>,
) -> Json<SimpleResponse> {
    let full_key = format!("{}:{}", namespace, key);
    let deleted = if state.engine.delete(&full_key) { 1 } else { 0 };
    Json(SimpleResponse { ok: true, deleted: Some(deleted) })
}

async fn delete_namespace(
    State(state): State<Arc<AppState>>,
    Path(namespace): Path<String>,
) -> Json<SimpleResponse> {
    let prefix = format!("{}:", namespace);
    let deleted = state.engine.delete_prefix(&prefix);
    Json(SimpleResponse { ok: true, deleted: Some(deleted) })
}
```

```rust
use crate::store::engine::KvEngine;
use crate::store::memory_type::MemoryType;
use crate::resp::parser::RespValue;
use std::sync::Arc;

pub struct CommandHandler {
    engine: Arc<KvEngine>,
}

impl CommandHandler {
    pub fn new(engine: Arc<KvEngine>) -> Self {
        Self { engine }
    }

    pub fn handle(&self, cmd: &crate::resp::parser::Command) -> RespValue {
        match cmd.name.as_str() {
            "PING" => RespValue::SimpleString("PONG".into()),
            "SET" => self.handle_set(&cmd.args),
            "GET" => self.handle_get(&cmd.args),
            "DEL" => self.handle_del(&cmd.args),
            "MGET" => self.handle_mget(&cmd.args),
            "MSET" => self.handle_mset(&cmd.args),
            "KEYS" => self.handle_keys(&cmd.args),
            "INFO" => self.handle_info(),
            "MSETT" => self.handle_msett(&cmd.args),
            "MGETT" => self.handle_mgett(&cmd.args),
            "MSCAN" => self.handle_mscan(&cmd.args),
            "MTYPE" => self.handle_mtype(&cmd.args),
            _ => RespValue::Error(format!("ERR unknown command '{}'", cmd.name)),
        }
```

```rust
pub mod store;
pub mod http;
pub mod resp;
pub mod config;

use clap::Parser;
use std::sync::Arc;
use tracing::info;

#[derive(Parser, Debug)]
#[command(name = "basalt", version, about = "Ultra-high performance KV store for AI memory")]
struct Args {
    /// HTTP host
    #[arg(long, default_value = "127.0.0.1")]
    http_host: String,

    /// HTTP port
    #[arg(long, default_value_t = 7380)]
    http_port: u16,

    /// RESP host
    #[arg(long, default_value = "127.0.0.1")]
    resp_host: String,

    /// RESP port (Redis-compatible)
    #[arg(long, default_value_t = 6380)]
    resp_port: u16,

    /// Number of shards
    #[arg(long, default_value_t = 64)]
    shards: usize,
}

#[tokio::main]
async fn main() {
    tracing_subscriber::fmt()
        .with_env_filter("basalt=info")
        .init();

    let args = Args::parse();
    let config = config::Config {
        http_host: args.http_host,
        http_port: args.http_port,
        resp_host: args.resp_host,
        resp_port: args.resp_port,
        shard_count: args.shards,
    };

    let engine = Arc::new(store::engine::KvEngine::new(config.shard_count));

    info!("basalt v0.1.0 — memory that moves fast");
    info!("shards: {}", config.shard_count);
    info!("HTTP:  {}:{}", config.http_host, config.http_port);
    info!("RESP:  {}:{}", config.resp_host, config.resp_port);

    // Run both servers concurrently
    let http_engine = engine.clone();
    let resp_engine = engine.clone();
    let http_config = config.clone();
    let resp_config = config.clone();

    let http_server = tokio::spawn(async move {
        let app = http::server::app(http_engine);
        let addr = format!("{}:{}", http_config.http_host, http_config.http_port);
        let listener = tokio::net::TcpListener::bind(&addr).await.unwrap();
        info!("HTTP listening on {}", addr);
        axum::serve(listener, app).await.unwrap();
    });

    let resp_server = tokio::spawn(async move {
        if let Err(e) = resp::server::run(
            &resp_config.resp_host,
            resp_config.resp_port,
            resp_engine,
        ).await {
            eprintln!("RESP server error: {}", e);
        }
    });

    tokio::select! {
        _ = http_server => info!("HTTP server stopped"),
        _ = resp_server => info!("RESP server stopped"),
    }
}
```
